    Developer's Documentation

    This documentation provides information for developers who would like to contribute to the project. It tries to explain how modules are organised and how PySide2 and Qt3D are used.

    The simulator is developped in Python 3.8+, using Pyside2, the Qt for Python package.

    The 3D view is based on the Qt3D framework. To have a better understanding of the code, it is important to understand Qt concepts like Signals and Slots, and the Qt3D Entity Component System (ECS).
